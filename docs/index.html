What is this?

This dashboard tracks a real-time mean-reversion model. It calculates the “fair” score of an ongoing NBA game and alerts us when the bookmaker's live betting line is significantly off-base.

  • Forward Test — Real bets the system has actually identified during live games.
  • Batch Backtest — How the model would have performed if it had been running on past playoff games.
  • Thresholds — Different “sensitivity” settings for the engine, balancing the number of bets taken vs. accuracy.
