# Idea Pipeline

*The backlog is the moat — not today's piece. A standing list of evergreen ideas worth writing, each
chosen because it's durable, not because a calendar needs filling. Titles only; a title earns a draft only
when it clearly passes the four tests in `Investment_Writing_Principles.md`. Over years this list becomes
intellectual capital in its own right.*

**Status key:** ✅ published · ✍️ drafted (held in reserve) · ○ idea
