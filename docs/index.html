What is this?

This dashboard tracks a real-time mean-reversion model for MLB. It calculates the “fair” run total of an ongoing game and alerts us when the bookmaker's live total is significantly off-base.

  • Forward Test — Real bets the system has actually identified during live games.
  • Batch Backtest — How the model would have performed if it had been running on past games.
  • Thresholds — Different “sensitivity” settings, balancing the number of bets taken vs. accuracy.
