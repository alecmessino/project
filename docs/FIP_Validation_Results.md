# Frog-in-the-Pan (Information Discreteness) — Validation Results

**Status: NOT VALIDATED — real-data run could not complete in this environment.**
This document records an *attempt* to validate the Frog-in-the-Pan (FIP) effect on Driftwood's
universe. The numbers below come from the **synthetic** harness only and are **not evidence** that
the effect exists. The Information-Discreteness signal remains an internal R&D artifact and is **not
wired into the live trading signal.**

---

## What was run

```
ID_SWEEP_REAL=1 python scripts/id_sweep.py
```

The script attempts to pull ~40 years of proxy-spliced daily history for the real 18-ETF MATRIX
(`drift.universes.MATRIX`) and, failing that, falls back to a deterministic synthetic universe.

**Outcome: the real-data pull returned 0 of 18 names** and the run fell back to synthetic. The
sandbox's outbound proxy allowlist covers only package registries (pypi/npm/etc.) — the market-data
hosts (stooq, Yahoo) are not reachable, so `pull_symbol("IVV", …)` returns 0 bars. **The real
validation must be run in an environment with market-data egress** (see "How to validate for real").

## Method

At each monthly rebalance (`rebalance_bars = 21`), over the cross-section:
1. Rank by the live volatility-normalized momentum score (`signal.momentum_score`, lookback 60).
2. Take the long half (positive score).
3. Split those longs at the **median Information Discreteness** (`signal.information_discreteness`):

   `ID = sign(PRET) × (%neg − %pos)` — low/negative ID = **continuous** information (the path the
   paper finds keeps trending); high ID = **discrete** jumps.
4. Measure each name's forward return to the next rebalance.

FIP predicts the **continuous** bucket out-returns the **discrete** bucket (a positive
continuous − discrete spread). Significance is a Welch two-sample t on that spread.

## Result (SYNTHETIC universe — mechanism check only, NOT evidence)

| Long bucket            | Mean fwd return | Info ratio | Obs  |
|------------------------|----------------:|-----------:|-----:|
| Continuous (low ID)    |          1.505% |      0.499 | 2612 |
| Discrete (high ID)     |          1.543% |      0.481 | 1681 |
| All longs (baseline)   |          1.520% |      0.491 | 4293 |

- **Continuous − Discrete spread:** **−0.038%** per 21-bar hold (FIP predicts *positive*).
- **Welch t-statistic:** **−0.39** → **not statistically significant** (|t| < 1.96).
- This t already *overstates* significance: forward returns share market moves cross-sectionally,
  which inflates the t — read it as a generous upper bound.

## Statistical interpretation

On the synthetic data the spread is **economically negligible and statistically insignificant**
(t = −0.39, wrong sign). This is the *expected* outcome: the synthetic series encodes different
return *paths* but not the *behavioral* mechanism (investor inattention to gradual news), so there
is nothing for the bucket split to detect. **It neither confirms nor refutes FIP** — it only proves
the harness discriminates the buckets and computes the statistic.

## Why transfer to this book is doubtful (the skeptic's case)

- **Documented on single stocks**, not ETFs. The effect rides on investor inattention to *firm-level*
  information; an ETF is a basket whose return path is a weighted average that **averages those
  idiosyncratic paths away**.
- **The universe is tiny.** 18 ETFs barely supports a median split, let alone the ID quintiles the
  paper uses on thousands of names — statistical power is minimal.
- **Cross-sectional correlation** among 18 broad ETFs further erodes the effective sample.

## Verdict

**FIP / Information Discreteness is UNVALIDATED for the Driftwood universe.** Do not present it to
clients or label anything "The Institutional Edge." It stays an internal research artifact, not in
the live signal.

Run, in an environment with market-data access:


Promote it only if the real run shows a **positive, statistically significant, and persistent**

1. a positive spread with |t| ≥ 2 on real 40-year history,
2. robustness across sub-periods (e.g. per-decade), not one regime,
3. survival of realistic costs and the small-universe / cross-correlation caveats above.
